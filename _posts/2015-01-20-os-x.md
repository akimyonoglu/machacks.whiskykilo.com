#### Stop the Hostname Changes

OS X renames itself a lot. It drives me nuts, so here's how to stop that behavior.

{% highlight bash %}
sudo scutil --set HostName bobjonescomputer
sudo echo "HOSTNAME=bobjonescomputer">>/etc/hostconfig
sudo reboot
{% endhighlight %}
